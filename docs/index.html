Look impossibly busy while you're away from the keyboard.

Built to look exactly like the real thing

Every detail borrows from a genuine terminal — the timing, the chrome, the phosphor. It's convincing enough that people will ask what you're compiling.

layout

1–8 live windows

Arranged in automatic tiled grids, overlapping cascades, or a mixed floating layout — reshuffled each time it starts.

animation

Typewriter timing

Commands are typed character by character with natural hesitation, then output streams in line by line before the session loops.

effects

Full CRT treatment

Scanlines, phosphor glow, subtle power-flicker, and a soft vignette — the whole retro-monitor package, all toggleable.

Running in under a minute

Build it yourself for the smoothest install, tap it with Homebrew, or grab a pre-built release.

Build from source

recommended

No Gatekeeper prompt — locally built bundles aren't quarantined. Needs Xcode Command Line Tools.

$ git clone https://github.com/yemzikk/mac-terminal-screen-saver.git $ cd mac-terminal-screen-saver $ make install

Homebrew tap

One command once the tap is published. --no-quarantine is needed (ad-hoc signed, not notarized).

$ brew tap yemzikk/tap $ brew install --cask \ --no-quarantine terminal-screensaver

Download release

Grab the .dmg or .zip, open the bundle, then click Install.

# after downloading: $ xattr -dr com.apple.quarantine \ ~/Downloads/TerminalScreenSaver.saver

Heads up on Gatekeeper. Released bundles are ad-hoc signed but not notarized (no paid Apple Developer account), so downloaded copies are quarantined by macOS. Clear it with the xattr command above, or just build from source to skip it entirely. Then open System Settings → Screen Saver and pick Terminal Screen Saver.
